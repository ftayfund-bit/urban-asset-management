# Urban Asset Management

Real estate development site for Urban Asset Management Bremen.

## About

Urban Asset Management specializes in transforming underutilized urban properties into thriving residential and mixed-use developments. This repository contains the website for UAM's flagship development project, The Bulwark, located in Bremen's UNESCO World Heritage old town.

## Project Details

- **Location**: Am Wall 117-125, Bremen Old Town
- **Type**: Office to Residential Conversion
- **Development Focus**: Premium residential communities in historic German city centers
